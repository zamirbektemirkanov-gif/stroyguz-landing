# СтройГруз — Лендинг для компании по грузоперевозкам

Пет‑проект: адаптивный одностраничный лендинг.

## Стек
- HTML5 (семантика, BEM)
- SCSS (переменные, миксины, адаптив)
- JavaScript ES6+ (калькулятор, валидация формы, анимации)
- Gulp (сборка, минификация)

## Функционал
- Адаптив: 320px / 768px / 1024px / 1440px
- JS‑калькулятор стоимости перевозки
- Валидация формы заявки
- Анимация блоков при скролле (IntersectionObserver)
- Якорное меню с плавным скроллом
